== Installation ==

= ssh into your server and clone the plugin (recommended);

* ssh mywebiste.com
* cd ./mywebiste.com/wp/wp-content/plugins/
* git clone git://github.com/etozzato/headcounters.git

= download the archive at http://github.com/etozzato/headcounters/archives/master and upload it to your plugins folder.
